Скрипты для первичной тестовой разработки, например:

1) создание объектов базы - таблицы, схемы, функции
2) перенос данных
3) трансформация данных
4) инициализация данных

/* как расчитывается сателит */
1) при дублях из загрузки берется максимальный датафлоу dttm
2) сравнение на измененные записи - join rk ников и сравнение хешей
3) добавление новых записе
4) поиск удаленных записей
5) обновление флагов актуальностей - запись актуальна, если среди таких же rk имеет макимальный dataflow_dttm
